# pipe-fatigue-damage-learning
本仓库用于记录管道疲劳损伤（Pipe Fatigue Damage）的系统性学习与代码实践。内容涵盖：1. 经典疲劳与断裂力学理论（S-N曲线、Coffin-Manson关系、雨流计数法与Miner规则）；2. 三维建模与有限元仿真分析（SolidWorks / Abaqus）；3. 探索结合科学机器学习（SciML, 如 DeepONet / PINNs）在疲劳损伤场演化与预测中的前沿应用。
本项目用于记录个人在管道疲劳损伤方向的系统性学习、有限元仿真以及算法开发过程。

## 研究方向
力学理论：高周/低周疲劳机理、线弹性断裂力学（LEFM）、损伤累积理论。
CAE 仿真复杂管件（如三通管、弯头）的应力集中分析与随机载荷下的寿命预测。


## 仓库目录

- `01_Theory_Notes/`：疲劳与断裂力学理论学习内容笔记。
- `02_CAD_Models/`：SolidWorks 建立的几何模型。
- `03_CAE_Simulations/`：有限元分析与结果后处理。


## 技术栈
- 编程语言：Python, C++
- 建模与仿真：SolidWorks, Abaqus, OpenFOAM
- 深度学习：PyTorch，tensorflow
